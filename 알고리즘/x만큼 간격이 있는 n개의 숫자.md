# x만큼 간격이 있는 n개의 숫자

### 문제 설명

함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

제한 조건

- x는 -10000000 이상, 10000000 이하인 정수입니다.
- n은 1000 이하인 자연수입니다.

### 문제 풀이

n번 만큼 반복해야 하기 때문에 i = 1 부터 n번까지 조건을 준다. 인덱스를 1부터 시작하는 이유는 x가 x씩 증가하기 때문에 i를 곱해주기 위해서다.

```js
function solution(x, n) {
  var answer = [];
  for (let i = 1; i <= n; i++) {
    answer.push(x * i);
  }
  return answer;
}
```
